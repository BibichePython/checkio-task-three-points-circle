Through any three points that do not exist on the same line, there lies a unique circle. The points of this circle are represented in a string with the coordinates like so:

    "(x1,y1),(x2,y2),(x3,y3)"

Where x1,y1,x2,y2,x3,y3 are integers.

You should to find the circle for three given points, such that the circle lies through these point and return the result as a string with the equation of the circle. In a Cartesian coordinate system (with an X and Y axis), the circle with central coordinates of (a,b) and radius of r can be described with the following equation: 

    "(x-x0)^2+(y-y0)^2=r^2"

where x0,y0,r are decimal numbers with a precision of two digits.

Conditions and remarks:
Given points does not lie at one line.
Remove extraneous zeros and all decimals, they are not necessary.
For rounding use the standard mathematical rule.

Input: A string with coordinates.

Output: A string. An equation of the circle.

Example: 

checkio("(2,2),(4,2),(2,4)") == "(x-4)^2+(y-4)^2=2.83^2"
checkio("(3,7),(6,9),(9,7)") == "(x-6)^2+(y-5.75)^2=3.25^2"

How it is used: Mathematical software.
